# Finanzas Simulador

Este es un simulador de créditos basado en el sistema **MiVivienda** para ayudar a los usuarios a calcular diferentes métricas financieras (como **TIR**, **VAN**, **TCEA**, etc.) relacionadas con los préstamos de vivienda.

## Tecnologías Utilizadas

- **React** (Frontend)
- **TypeScript**
- **Vite** (Bundler)
- **TailwindCSS** (Estilos)
- **Node.js** (Backend, si se requiere)
- **MySQL** (Base de datos)
